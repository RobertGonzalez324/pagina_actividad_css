Proyecto web estático

Este proyecto es un sitio de 4 páginas en HTML y CSS con una temática inspirada en Assassin's Creed.

Páginas:
- index.html
- nosotros.html
- servicios.html
- contacto.html

Para ver el proyecto abre index.html en el navegador o abre cada página directamente.

La página de contacto usa localStorage para guardar el acceso localmente. No necesita backend ni servidor externo.

Los archivos principales son:
- css/estilos.css
- js/main.js

El proyecto está listo para usarse en local y para entregar.
