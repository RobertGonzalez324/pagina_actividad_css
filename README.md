Proyecto web estático

Este proyecto es un sitio de 4 páginas en HTML y CSS con diseño responsive, menú desplegable, carrusel de imágenes, acordeón y efecto parallax.

Páginas:
- index.html
- nosotros.html
- servicios.html
- contacto.html

Características principales:
- menú responsive y desplegable
- carrusel de imágenes en la página principal
- efecto parallax en el hero al hacer scroll
- acordeón en servicios
- diseño adaptado a móvil y escritorio
- Webpack para minificar HTML y CSS

Instalación y compilación:
1. Abre la carpeta del proyecto.
2. Ejecuta `npm install`.
3. Ejecuta `npm run build`.
4. Abre `dist/index.html` en el navegador.

También puedes usar las páginas originales abriendo los archivos `index.html`, `nosotros.html`, `servicios.html` y `contacto.html` directamente.

Los archivos principales son:
- css/estilos.css
- js/main.js
- js/app.js
- webpack.config.js
- package.json
